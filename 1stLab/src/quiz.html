How web browser and web server work

Web Browser

Web Browser main function is to receive and present the information from web resource chosen by user. Browser request the data from server via HTTP Request. The resource is usually an HTML document, but may also be in form of PDF, Image, or other suitable type. Location of the resource is specified and input by the user using a URI (Uniform Resource Identifier).

Web Server

Web server main function is to listen to the HTTP request around the world and handle them. Web server is a program that use HTTP to serve the file from server to client response to thier requests.
